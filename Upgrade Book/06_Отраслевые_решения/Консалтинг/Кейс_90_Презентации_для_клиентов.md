# Кейс №90: Презентация решения за 30 минут

Нужна презентация для клиента, дизайнер занят, делаете сами 3 часа, выглядит плохо.

## Было

Консультант готовил презентацию стратегии для клиента. Собирал 3 часа: структура, тексты, оформление. Получалось скучно, клиент не впечатлялся.

## Решение

Создаёт структуру и тексты в ChatGPT за 20 минут, оформляет в Canva по шаблону за 10 минут. Презентация выглядит профессионально, клиенты довольны. Время сократилось с 3 часов до 30 минут.

## Инструменты

**ChatGPT** (структура и тексты)
**Canva/Gamma** (дизайн)

## Промт

```
Создай структуру и тексты презентации для клиента.

ТЕМА: [что презентуем]
ЦЕЛЬ: [убедить / объяснить / продать]
АУДИТОРИЯ: [кто будет смотреть]
ВРЕМЯ ПРЕЗЕНТАЦИИ: [минут]

СТРУКТУРА:

Слайд 1: Обложка
Название проекта, для кого, дата

Слайд 2: Проблема
Что беспокоит клиента сейчас (боли, цифры)

Слайд 3: Решение
Наш подход в 1 абзаце

Слайды 4-8: Детали решения
Этапы, инструменты, сроки, ресурсы

Слайд 9: Результаты
Конкретные метрики которые получит клиент

Слайд 10: Кейсы
Примеры из практики (если есть)

Слайд 11: Стоимость и сроки
Прозрачно

Слайд 12: Следующие шаги
Что делаем завтра

ДЛЯ КАЖДОГО СЛАЙДА: заголовок + 2-3 пункта текста (не больше).
```

## Результат

Презентация готова за 30 минут вместо 3 часов. Экономия 2,5 часа на каждую. Качество выше — структура логичная, дизайн профессиональный. Конверсия презентаций в сделки +20%.

## Как повторить

1. Опишите суть проекта и аудиторию
2. Сгенерируйте структуру в ChatGPT
3. Скопируйте тексты слайдов
4. Откройте Canva, выберите шаблон презентации
5. Вставьте тексты, добавьте иконки/графики
6. Экспортируйте в PDF/PPT

## Лайфхак

Используйте Gamma.app — загружаете текст от ChatGPT, ИИ автоматически создаёт презентацию с дизайном. Готово за 5 минут.

## Ограничения

Презентация — это упаковка. Если суть слабая, красивые слайды не спасут. Сначала сильное решение, потом презентация.
